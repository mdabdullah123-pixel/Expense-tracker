# Spec-Kit Setup

Place feature specs in the `specs/` directory using the spec template.

Workflow
--------
1. Create a spec in `specs/` using `.specify/templates/spec-template.md`.
2. Create a plan and tasks using the templates in `.specify/templates/`.
3. Link the spec to issues/PRs and reference the spec in commits.

Local tooling
--------------
This repository uses a simple file-based Spec-Kit; no additional tooling is required.
