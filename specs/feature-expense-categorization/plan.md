# Expense Categorization Plan

## Goal

Implement AI-assisted expense categorization to improve user workflow and reduce manual entry.

## Steps

1. Update `AIService` to include `categorize_expense()`.
2. Add UI suggestion handling in `pages/expenses.py`.
3. Validate category suggestions against allowed categories.
4. Write tests for category suggestion flow.

## Timeline

- Day 1: Design API and update service layer
- Day 2: Add UI integration and validation
- Day 3: Test coverage and review

## Risks

- Incorrect category suggestions may confuse users.
- Missing AI provider credentials may disable the feature.
